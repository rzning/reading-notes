区块链2.0 : 以太坊应用开发指南
==========================

<BookInfo/>

本书是一本用于指导区块链技术开发人员从事以太坊技术开发的系统性开发指导书籍。

本书包括：

- 区块链与以太坊的基本概念、相互关系，
- 以太坊的软件架构、工作机理，
- 以太坊的安装、网络配置，开发环境搭建，
- 以太坊的几类接口方式，特别是应用开发接口方式，
- 以太坊智能合约概念、开发、部署，
- 最后重点介绍了以太坊最经典的三类智能合约应用案例。
