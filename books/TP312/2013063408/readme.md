JavaScript 设计模式
==================

<BookInfo/>

本书通过介绍实用的 15 种 JavaScript 设计模式，
教会读者如何编写优美、结构型好和可维护的 JavaScript 程序，
以及如何将经典的和现代的设计模式应用到 JavaScript 语言中。

本书将带领你了解现代模块格式、如何有效为代码定义名称空间和其它重要主题。
