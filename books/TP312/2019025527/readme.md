TypeScript 图形渲染实战
=====================
2D 架构设计与实现
---------------

本书使用微软最新的 TypeScript 语言，以面向接口及泛型的编程方式，
采用 HTML 5 中的 Canvas2D 绘图 API，实现了一个2D动画精灵系统，
并在该精灵系统上演示了精心设计的与图形数学变换相关的 Demo。
